# customer-segment

## Khởi tạo & kích hoạt môi trường nào

1. Truy cập folder customer-segment

   `cd  customer-segment`

2. Tạo virtual env

   `python -m venv venv`

   `venv\Scripts\activate`   # Windows

   `source venv/bin/activate`  # Mac/Linux

3. Cài thư viện

   `pip install -r requirements.txt`
